# Forge Scout

OutSystems Forge component intelligence — natural-language search, license clarity, and GitHub repo health signals.

Built with React + TypeScript outside the OutSystems platform to demonstrate full-stack capability while solving a real OutSystems community problem.

## Planned Stack

- **TanStack Start** (Vite + React + TypeScript) — fullstack framework
- **TanStack Router / Query / Form / DB** — routing, server state, forms, client store
- **Tailwind CSS + shadcn/ui** — styling and components
- **PostgreSQL + Drizzle ORM** — data layer
- **OpenRouter** — LLM (free-tier models)
- **Biome** — linting and formatting

## Status

Scaffold in progress. Feature work lands incrementally through PRs — see the [open pull requests](../../pulls) for current state.
